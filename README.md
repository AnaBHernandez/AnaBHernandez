## 🌈 Método MIMO

El método MIMO surge de la filosofía de que "tu casa es tu mente", conectando la organización de espacios físicos con el bienestar mental. Esta metodología estructura las tareas en cuatro dimensiones fundamentales:

- **Mírate**: Dimensión de introspección y autoconocimiento
- **Imagina**: Dimensión de creatividad y planificación
- **Muévete**: Dimensión de acción y ejecución 
- **Ordena**: Dimensión de estructura y optimización

La aplicación digitaliza esta metodología original, permitiendo a los usuarios categorizar sus tareas y visualizar el equilibrio entre las diferentes dimensiones de su vida.
